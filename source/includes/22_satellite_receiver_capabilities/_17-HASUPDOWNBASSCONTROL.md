## has\_up\_down\_bass\_control

Capable of directly specifying a bass setting.


### Signature

`<has_up_down_bass_control></has_up_down_bass_control>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_up_down_bass_control>true</has_up_down_bass_control>
</capabilities>
```
