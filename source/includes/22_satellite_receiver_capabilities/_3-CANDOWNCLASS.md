## can\_downclass

Specifies if the A/V switch can down convert. For example: S-Video -\> Composite


### Signature

`<can_downclass></can_downclass>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <can_downclass>true</can_downclass>
</capabilities>
```
